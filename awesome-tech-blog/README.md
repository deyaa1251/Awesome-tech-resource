- [The Morning Paper](https://blog.acolyer.org/) - Daily review of CS papers (concluded, extensive archive).

## Contributing

Contributions are welcome and encouraged! This list thrives on community input.

### How to Contribute

1. Fork this repository
2. Add your resource following the format below
3. Ensure the resource is high-quality and actively maintained
4. Submit a pull request

### Contribution Guidelines

- **Relevance**: Resources must be related to tech learning and education
- **Quality**: Only include high-quality, well-maintained resources
- **Format**: Follow the existing format: `- [Name](url) - Description.`
- **Alphabetical Order**: Add resources alphabetically within their category
- **Working Links**: Verify all links are functional before submitting
- **One Per PR**: Submit one resource per pull request for easier review
- **No Duplicates**: Check that the resource isn't already listed
