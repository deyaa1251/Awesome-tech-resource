# Contributing to LLMObs Core

Thank you for your interest in contributing to LLMObs Core!

## Development Setup

1. Clone the repository
2. Install dependencies:
   ```bash
   make install
   ```
3. Start development services:
   ```bash
   make start
   ```

## Development Workflow

1. Create a feature branch
2. Make your changes
3. Run tests: `make test`
4. Submit a pull request

## Code Style

- Python: Follow PEP 8
- TypeScript/JavaScript: Use ESLint configuration
- Write meaningful commit messages

## Testing

- Write tests for new features
- Ensure all tests pass before submitting PR
- Maintain or improve code coverage

## Plugin Development

See the plugin development guide in the docs folder for creating new plugins.
